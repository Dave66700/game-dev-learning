# 199. 二叉树的右视图

给定一个二叉树的 根节点 root，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

BFS、DFS遍历顺序回顾

```
        1
       / \
      2   3
       \   \
        5   4
```

**1、BFS——层序遍历**
- 访问第一层1
- 第二层2-3
- 第三层5-4
- 1-2-3-4-5
**2、DFS——前序遍历 (根-左-右)**
这里是DFS前序遍历的变体，执行顺序根 -> 右 -> 左。
- 访问根节点
- 右子节点3（深度1）
- 访问3的右边子节点4（深度为2）
- 4没有子树。从而回溯到3，3没有左子节点，回溯到1
- 访问1的左子节点2
- 访问2的右边子节点5
- 1-3-5-2-5

**BFS方法**

1. 定义结果数组，用于存放每一层最右边的节点值
2. 边界条件：如果根节点为空，直接返回空结果
3. 核心数据结构：队列，用于进行层序遍历
4. 开始层序遍历：当队列不为空时，说明还有节点未处理
5. 所有层处理完毕，返回结果
