## 双指针算法总结：

**双指针是一种使用两个指针（索引）在数组或链表中协同工作的算法技巧。**


## 三种指针类型：

**1.对撞指针**
```
int left = 0, right = n - 1;
while (left < right) {
    // 根据条件移动 left 或 right
}
```

两个指针从两端向中间移动

应用：由序数组的两数之和、乘水最多容器


**2.快慢指针**

```
int slow = 0;
for (int fast = 0; fast < n; fast++) {
    if (条件) {
        // 处理 nums[slow]
        slow++;
    }
}
```
两个指针，一个指针快，一个指针慢

应用：移除元素，删除有序数组重复项

**3.滑动窗口（待会儿单独拿出来了）**

```
int left = 0;
for (int right = 0; right < n; right++) {
    // 扩展右边界
    while (窗口不满足条件) {
        // 收缩左边界
        left++;
    }
    // 更新结果
}
```

特点：维护一个区间，根据条件滑动

## 双指针解决问题的思路

- 第一步：确定指针含义
  - 每个指针代表什么？ 
- 第二步：确定初始位置
  - 指针从哪里开始？
  - 一般是两端开始、同时（快慢指针）开始 
- 第三步：确定移动条件
  - 关键问题：什么时候移动左指针？什么时候移动右指针？
  -  **移动原则：总是移动可能带来更好结果的那一侧**
- 第四步：确定终止条件
  - 对于两端开始： 一般是 left < right 或 left <= right
  - 对于快慢指针：fast 指针遍历完数组
