# 摄像机平滑跟随（Smooth Follow）

## 为什么会抖？

角色位置 = 新位置

摄像机位置 = 立刻等于目标位置

也就是——瞬移跟随；

**游戏里解决：**

让摄像机慢慢追上目标

这个技术叫做：**插值**

## 最常用的一种方式

vector3.Lerp();

不是一步到位，从A点往B点靠近一点

## 修改对象？

不是CameraController，**而是camera本体的位置跟随逻辑**

## 现在为什么不抖了？

 我们要打破player移动，由于camera是他的子物体就会一同被牵制的刚性绑定

 改成：
 Player（移动）

CameraTarget（只负责旋转）

Camera（独立存在，通过代码跟随 CameraTarget）

曾经是： 位置跟随 = 层级关系
```Player
└── CameraTarget
    └── Main Camera   （问题就在这里）
```

现在是： 位置跟随 = 数学计算
```Player
└── CameraTarget

Main Camera   （独立物体）
```
