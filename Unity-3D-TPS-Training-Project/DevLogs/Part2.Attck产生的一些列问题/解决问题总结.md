## 前面遇到的 3D 问题时可能有**三大类**错误：

| 类别      | 症状      |解决的方式|
| ------- | ------- | -------- |
| 方向错误    | 线打歪     |  已排除    |
| 物理检测问题  | 看得到却打不到 |两种不同的ray    |
| 数值/配置问题 | 改代码没反应  | 序列化问题  |

### 真实的原因

代码里改 attackRange 没用

Inspector 改了立刻命中

之前射线根本“够不到敌人”

不是方向问题，是 长度不够。

而你以为改了长度，其实没改到运行值（被 Inspector 覆盖）。

### 以后解决问题的方式，优先级：

①画线 → 看方向

② 拉长距离 → 看长度

③看 Collider / Layer

④才怀疑复杂逻辑
